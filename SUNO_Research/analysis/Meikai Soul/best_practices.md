# ベストプラクティス — Meikai Soul

## 1. ジャンルは「5〜7軸の過剰クロス」で書く

| こう書くと | こうなる |
|---|---|
| `Enka × Progressive J-metal × Indie Folk Rock × Thrash Metal × Speed Metal` | 5層の音像が立ち上がる重厚アンサンブル |
| `Heavy Metal × Hard Rock × Blues Rock × Psychedelic Funk × Industrial Gothic × Crust Punk × Denpa × 和楽器` | 8軸の闇広がりロック |
| `Britpop × Alt Rock × melodic distorted guitar × layered gang vocals` | 90sアンセム |

→ TenJR/CK/音色は2-3軸が主流だが、Meikai Soulは**5軸以上で破綻させない**のが武器。

## 2. シネマ的アレンジ進行を文章で書く

```
Verses crawl with fragile, near-broken female lead
Pre-chorus builds with low chants, tom pulses, rising urgency
Chorus explodes into fast thrash/speed intensity with double-kick drums
Bridge drops to steel guitar drones and cracked voice
Rebuilding into an explosive final chorus
```

→ kedarugeも進行記述するが、Meikai Soulはより**演奏ニュアンスの動詞 (crawl / explodes / rebuilding)** を使う。

## 3. 和楽器をロック楽器と同等に扱う

```
shamisen, koto, chiming guitars, taiko drums
Three shamisen playing simultaneously in contrasting rhythms
chaotic shakuhachi lines scream, bend, and spiral
```

→ shamisen/taiko/shakuhachi/koto を**装飾でなく主役級**として配置。

## 4. 「和楽器のみ」勝負も試す

```
A hyper-chaotic denpa-style Japanese track built around three shamisen ...
No guitars, no synths, pure controlled chaos
```

→ ロック楽器を全削除して和楽器のみで完結させる挑戦。糸の影方式。

## 5. ボーカル指定は曲ごとに完全別人格

| 曲のキャラ | ボーカル指定例 |
|---|---|
| 叙事詩ヒロイン | `fragile, near-broken female lead and distant young scream echoes` |
| グランジ男性 | `Raw, raspy vocals teeter between apathy and anger` |
| パンク怒号 | `Gritty, shouted male vocals tear through the mix` |
| Twin Vocal | `gritty, emotional male vocals deliver raw verses, while powerful soaring female vocals take the choruses` |
| Denpa kawaii | `denpa-style melodies, sudden pitch jumps, rapid-fire phrasing, playful madness` |
| Britpop nostalgic | `slight nasal tone, energetic emotional male vocal` |

→ **男性 / 女性 / Twin / 英日 / Denpa / 演歌**まで、楽曲ごとに完全別人格を指定する。

## 6. ネガティブは「装置限定」目的で使う

```
No guitars, no synths, pure controlled chaos     # 和楽器のみで成立させる
no synth, no modern pop                            # ヴィンテージ徹底
‑Electric Guitar, ‑Brass Sections                  # kedaruge方式 (短縮)
```

→ 「仕上げ拒否」(TenJR) ではなく、**特定楽器の意図的排除**で純度を上げる。

## 7. アナログ/ヴィンテージ語彙で温度を上げる

```
analog recording, vintage tone
raw yet polished analog warmth with subtle tube amp overdrive
Wide cinematic mix evolving from suffocating to radiant
```

→ デジタル先鋭さでなく**真空管+空気感**を志向すると温度が出る。

## 8. 歌詞は王道直球で書く

頻出語彙バンク:
- 夜 / 闇 / 光 / 朝 / 月
- 叫び / 声 / 鼓動 / 胸
- 諦めない / 越える / 立ち上がる / 越えていけ
- 痛み / 傷 / 涙 / 燃え尽きた
- 自由 / 真実 / 秘密 / 道

→ 難解な比喩より**90年代J-Rockの直球エモーション**で勝負。

## 9. 「夜 → 光」の三段アーク構造

```
Verse: 夜の苦難 (痛み/迷い/孤独)
   ↓
Pre-Chorus: 越境の決意
   ↓
Chorus: 光の到来 (朝/胸/輝き)
```

例 (夜を越えた光):
- 夜 → `眠れぬ夜 数えてた / 終わりのない 痛みの中`
- 越境 → `何度でも 崩れながら / それでもまだ 立ち上がる`
- 光 → `夜を越えた光が今 / この胸を 照らしてる`

→ 4-6曲で同じアークを反復。アーティスト固有の物語構造。

## 10. 反復フックを2-4回打つ

```
Step into the dark, 闇を追いかけ × 4
Don't wake me up × 3
It's the same dream every night × 3
```

→ 90年代スタジアムロック式の**徹底反復**。脳に刻む強度。

## 11. 設問形式で読者を巻き込む

```
何が君を　幸せにする？
If you're not real / Then why does it hurt?
これが生きることなら / 痛くても ここにいたい
```

→ ランジェリー的設問だが、Meikai Soulは**ロック的応答待ちの叫び**として使う。

## 12. Bridgeで視点反転

```
[Bridge]
If you're not real
Then why does it hurt?
If you're not mine
Then why does it work?
```

→ 「ない」の反復で疑念→ラスサビで肯定転換。テンションの放出装置。

## 13. コミカル路線も持つ勇気

```
[Final Line – 爆発]
夢だったあああああ！！！！！
(ヘイ！)
```

→ 真面目なロックばかりでなく、**ドタバタコメディロック**を1曲入れる。アーティストの人間味が増す。

## 14. NOTES欄に1行の自己評価を必ず残す

```
このひとの本気のギターは天才的！
三味線もすげえ
サビのメロディがすごい良い！
まさに理想的な和ロック。
```

→ 自作の評価ログ。後の解析や再生成判断に活かす。

## 15. 構成マーカーは「英日切り替え」で書ける

- 英語: `[Verse 1] [Chorus] [Bridge]`
- 日本語: `【Verse 1】【Chorus】【Bridge】`

→ 楽曲のジャンル/言語に合わせてマーカー記法を選ぶ。J-Rockなら【】、Britpopなら[]。

---

## Meikai Soul流の哲学

> **「ジャンルを限定するな。和洋を融合せよ。直球で殴れ。NOTES に感想を残せ。」**

特定ジャンルに絞るのではなく、ロックの全部を1人で回す総合派。武器は (1) 過剰ジャンル合成 (2) 和楽器の主役起用 (3) 王道直球の歌詞 (4) 多様なボーカル使い分け。

このアプローチが向く人:
- ロック全般が好きで特定ジャンルに絞れない
- 和楽器とロックの融合に興味がある
- 王道のエモーションを直球で書きたい
- 大量に曲を作って多様性を試したい
