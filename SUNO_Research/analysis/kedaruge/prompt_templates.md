## A. ラベル形式 (Style/Mood/Energy/Beat/Instrument/FX/Vocal)

### PROMPT欄
```
[Style: {ジャンル1, ジャンル2}]
[Mood: {Euphoric / Melancholic / Tragic / Warm / Mysterious}, {副情緒}]
[Energy: {Floating / Slow Build / Explosive Peaks}]
[Instrument: {Flute, Clarinet, Piccolo, Soft Pads, Sub Bass} 等]
[Beat: {Fast Breakbeat / Half-time / Abstract Slow Dubstep}]
[FX: {Reverb, Delay, Filter Sweeps, Glitch, Sparkle Textures}]
[Vocal: {Sweet, Light, Feminine / Fragile, Ethereal / High-pitched Sweet Female}]

{追加の文章説明: 楽曲の物語的進行、雰囲気、特殊な指示}
```

→ Journey, Oblivion, Flora, Hide & Seek 型。ラベルで分割すると整理しやすい。

## 共通: 歌詞構造テンプレ (kedaruge流)

```
[Intro]
{詩的命題 or 効果音表記 2-4行}
{(括弧呟き 任意)}

[Verse 1]
{科学/哲学用語を含む比喩 4-8行}
{童話的モチーフを並置}

[Pre-Chorus]
{抽象から具象への移行 2-4行}
{（括弧呟きで内的レイヤー）}

[Chorus]
{主語+は+抽象動詞 の同型連打 4-8行}
or
{詩的命題のリフレイン}
{ルビで多義化}

[Verse 2]
{Verse 1の対概念 or 視点反転}

[Bridge]
{哲学的問いかけ or 童話的場面転換}
{（括弧で囁き）}

[Chorus] (variation)
{Chorusの変奏 or 同じ命題の再提示}

[Outro]
{詩的余韻、記号化された終わり}
{シ。 / 消去。 / End. のような視覚効果}
```

## 重要: kedarugeを再現するチェックリスト

- [ ] プロンプトに音響工学用語 (granular / sidechain / FM / tape saturation等) を3つ以上入れたか
- [ ] 木管楽器 (flute / clarinet / oboe / bassoon) を最低1つ指定したか
- [ ] Sub bass / sub-bass を入れたか
- [ ] Silence / hard stops / dropouts のいずれかを指定したか
- [ ] 歌詞に科学・哲学用語を3つ以上比喩として埋め込んだか
- [ ] 童話モチーフ (お砂糖、三輪車、隠れ家、もういいかい？等) を最低1つ入れたか
- [ ] ルビで漢字に音楽的読みを当てたか
- [ ] 括弧書きの内的呟きを最低1ヶ所入れたか
- [ ] 構造マーカーは半角 [Verse 1] [Chorus] にしたか
- [ ] アーティスト固有モチーフ (軌道 / 螺旋 / 鳥籠 等から自前で1つ選定) を反復したか
