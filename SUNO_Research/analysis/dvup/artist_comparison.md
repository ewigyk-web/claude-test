## dvupのクセ・強み

強み:
- **メタル/V系の網羅性**: Symphonic / Power / Death / V系 / Math Core / Loli Pop / Middle Eastern まで、メタル系統を多角的にカバー。
- **参照アーティストの使い分け**: X JAPAN / BABY METAL / 黒夢 / 90年代V系金字塔 を意識的に切り替える。Daraの単一目標 (米津) と異なる**多数派目標**戦略。
- **「ライブ感タグ + ジャンル本体」の2段プロンプト**: ライブ感を最初に確立してからジャンル詳細を書く独自構造。
- **ボーカル4タイプの使い分け**: 男性デス / 男性クリーン高音 / 女性高音 / 女性Sweet+Scream を曲ごとに切替。
- **英語メタル歌詞の精度**: メタル王道語彙 (Fire/Flame/Wings/Eternity 等) を効果的に使う。
- **V系日本語の耽美漢字**: 「紅い花」「枯れた華」「淫乱」「虚無」など黒夢/Dir en grey系の語彙。
- **異色実験**: Metal Tiger (ロリポップ×メタル), SUNSCREEN FAILURE (中東×ブラックユーモア) など、メタル枠内で実験。
- **元バンドボーカル**としての本人ボイス活用 (緋ト蒼)。

クセ:
- メタル/V系から外れない → 多様性は高いが**枠外への展開は無い**。
- 英語比率が高い → 日本語曲 (ギルティ等) は限定的。
- BPM範囲指定が多い → 単一値より揺らぎが大きい。
- ネガプロほぼ無し → AIが想定外の方向に振れる可能性。
- 自賛系NOTES (「超名曲」「クソ上手い」) → 客観評価には注意。

## dvupを位置付ける一行

> **「メタル/V系の全時代を網羅するコレクター。X JAPANから黒夢まで参照を切り替える。」**

Daraが単一アーティスト (米津玄師) を目標にするのに対し、dvupは**メタル/V系全体**を視野に入れて、曲ごとに参照アーティストを切り替える。X JAPAN / BABY METAL / 黒夢 / 90年代V系金字塔 / 御本人ボイス を**すべて使い分ける**ジャンル特化型コレクター。

このアプローチが向く人:
- メタル/V系のあらゆるサブジャンルが好き
- 特定アーティストではなく**ジャンル全体**を再現したい
- 英語メタル歌詞を書きたい
- V系の耽美漢字も使いこなしたい
- 自分の音楽的引き出しの広さを示したい

→ **Dara=単一模倣**、**dvup=ジャンル特化コレクター**、**Meikai Soul=和洋融合総合**の三者三様。
