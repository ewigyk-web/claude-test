## 共通: 歌詞構造テンプレ (Meikai Soul流 — 王道アーク)

```
[Intro] (任意)
{場面設定 or 効果音 2-4行}

[Verse 1]
{夜の苦難・迷い・孤独 8行}
{4行ごとにシーンを切り替える}

[Pre-Chorus]
{越境の決意 2-4行}
{反復で緊張を上げる}

[Chorus]
{タイトル/キーフレーズの直球宣言}
{光・朝・胸・鼓動 等の語彙}
{タイトル反復 (2-4回)}

[Verse 2]
{Verse 1の変奏 or 視点強化}

[Pre-Chorus 2]
{緊張のさらなる上げ}

[Chorus 2]
{Chorusの変奏}

[Bridge]
{視点反転 or 設問}
{2行 × 2回の対句構造が有効}

[Final Chorus]
{Chorusの最大拡張版}
{追加の救済フレーズを盛り込む}

[Outro]
{静かな余韻 or 短い宣言}
```

## 重要: Meikai Soulを再現するチェックリスト

- [ ] プロンプトに5軸以上のジャンルクロスを書いたか
- [ ] アレンジ進行 (Verse→Pre→Chorus→Bridge→Final) を文章で記述したか
- [ ] ボーカル質感を最低3要素で指定したか (例: gritty, raspy, emotional)
- [ ] 和楽器を使うなら主役級として配置したか (装飾扱いにしない)
- [ ] アナログ/ヴィンテージ語彙を1つ以上入れたか
- [ ] 歌詞テーマを「夜→光」の王道アークに沿わせたか
- [ ] サビでタイトル/キーフレーズを2-4回反復したか
- [ ] Bridgeで視点反転 or 設問を入れたか
- [ ] 構成マーカーは英日どちらかに統一したか
- [ ] NOTES欄に自作の評価1行を残したか
