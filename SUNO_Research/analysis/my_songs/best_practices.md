# ベストプラクティス — Yuto (確立済みの手法)

Yutoが既に身につけている強みを整理。これは**今後も継続すべき手法**。

---

## 1. 「他人のプロンプトをパクる」研究的姿勢

NOTESに明記:
> 他人のプロンプトをパクる！！ (呑め日本晴れ)

→ 良い結果を出している他アーティストの手法を**素直に試す**姿勢。これが最大の成長エンジン。

## 2. 学習結果のNOTES記録

```
歌詞中のメタタグの一部は、とある巨匠の真似です。
メタタグ入れたり、プロンプトに音楽と関係ないワードを入れると、AIが本気出しやすくなる。
```

→ 学んだ手法を**自分のNOTESに残す**ことで、後の楽曲制作で参照可能にする。研究ノート的運用。

## 3. SUNOコミュニティへの自己言及テーマ

最大の発見ジャンル:
- LIKE LIKE LIKE (通知が来ない辛さ)
- 創造せえよ (AIに頼りすぎる自分への説教)
- 俺が作ったことにしたい (ユーザーの本音)
- 終日SUNO (SUNO中毒)
- 1000 Ears 1000 Eyes (フォロワー達成)

→ **SUNO ユーザーに刺さるテーマ**を発見済み。一番人気曲はここから出ている。

## 4. メタタグの歌詞内埋め込み

```
[year: 2050][date: February-31][location: Neo Shinagawa]
[loss-less audio][layered][wide dynamic range]
[melody complexity level: 1.1]
[rhythm complexity level: 1.5]
[chord complexity level: 1.7]
```

→ 「音楽と関係ないワードでAIが本気を出す」原理を実装。再現性のある手法として武器化。

## 5. シーン/場所記述

具体的な地名・時間を指定する:
- `Yokohama Minatomirai` (SORA)
- `Roppongi Hills at 1 am` (Come on Let's Dance)
- `shinjuku kabukicho in heavy rain` (雨がきた)

→ AIに**具体的情景を想起させる**ことで音像が立体化する。

## 6. ボーカル指定テンプレ

```
Female vocal: sharp, very sultry, decadent, exaggerated vibrato,
frequent kobushi turns, Rolled R consonants, sneering phrasing, playful sarcasm
```

→ 同じフレーズを複数曲 (混沌, 孤独は旧びたり, 終日SUNO, 呑め日本晴れ) で流用する**自分用テンプレ**を確立済み。

## 7. en-dash型ネガの定型化

```
‑shout, ‑scream, ‑growl
‑guitar solo, ‑shout, ‑scream
```

→ 何曲かで同じネガセットを使用。Daraほどではないが、**自分用の固定ネガ**を持っている。

## 8. 「もう一回」「ゼロから」のモチーフ反復

複数曲で同じテーマフレーズが現れる:
- 「もう一回 もう一回 もう一回」(Regeneration)
- 「ゼロから もう一度」(創造せえよ)
- 「再生 再生 再生」(LIKE LIKE LIKE)

→ アーティスト固有の**世界観モチーフ**として機能している。

## 9. 影響元アーティストの活用

NOTESで明示:
- DAITA / SIAM SHADE → Cloud OverDrive
- HYDE → 灰色のHEART, Tesoro y Colmillo
- UVERworld → Break Through, SORA
- HYDE × Co Shu Nie → Tesoro y Colmillo
- メガテラゼロ → Dear Listeners

→ **目標サウンドを明確に持って**プロンプトを書く Daraスタイルを部分的に採用。

## 10. ジャンル横断の実験力

8系統 (J-Rock / 和 / Rap-Rock / Future Bass / Jersey / Industrial / Britpop / Punk) を試している。
→ 1ジャンル特化型の他アーティスト (CK = Djent, dvup = メタル) と異なる**全方位実験家**としての強み。

## 11. REMIX文化の活用

8曲がREMIX。**他者の素材を再構築**することで:
- 学習効率が上がる (元曲の良さを分析)
- ネタ切れを防げる
- コミュニティとの繋がりが生まれる

→ オリジナル制作と並行する**第二の制作回路**として確立済み。

## 12. 多言語コードスイッチ実験

Tesoro y Colmillo の5言語混合 (日英西独露):
```
ゆらり thin skin, dim light クヨクヨ deep inside
まだ cold まだ sore Ich atme 青い棘
言えない no lo sé
ほどけない この fever но я жива 花の影
```

→ 他アーティストにない独自実験。**HYDE × Co Shu Nieのコラボ曲**を起点に、独自スタイルへ発展させる手法。

## 13. 自己卑下→発奮の歌詞アーク

eclipseの「劣等感→自己肯定」と同型の心理アーク:
- LIKE LIKE LIKE: 「通知なんて来やしない」 → 「音はここで光る」
- 創造せえよ: 「ゼロから作り方忘れた」 → 「もう一回 壁を壊せ」
- Break Through: 「一番深いどんぞこ」 → 「ぶち破れ」

→ **SNS時代の比較疲れ**を起点にする現代的テーマ。

## 14. 等身大の生活感を歌詞にする

- LIVEに行こうぜ (ライブ参加の現実)
- Welcome to my world (仕事漬け)
- 終日SUNO (有給SUNO中毒)
- 呑め日本晴れ (社畜と祭り)

→ 抽象的な詩ではなく**具体的な日常**を直接書く。共感を生む武器。

## 15. NOTES活用による戦略性

各曲のNOTESに:
- 制作意図 / 影響元アーティスト
- 反応 (再生数 / 一番人気 / Hooks一位)
- 学んだ手法
- イベントエントリー情報

→ **マーケティング志向**の記録術。

---

## まとめ: Yutoが既に持つ武器

1. **多ジャンル実験力** (8系統網羅)
2. **他アーティスト手法の吸収術** (10者から取り入れ)
3. **SUNOコミュニティ向け歌詞** (最大の差別化要素)
4. **メタタグ研究** (年代/場所/complexity)
5. **REMIX文化** (8曲)
6. **影響元の明確化** (HYDE/UVERworld/SIAM SHADE)
7. **多言語実験** (5言語混合)
8. **NOTES戦略記録** (制作ログ)
9. **シーン記述プロンプト** (Yokohama/Roppongi/新宿)
10. **モチーフ反復** (もう一回/ゼロから/再生)
