対象: `artists/ランジェリー/` 配下 12曲。★付きなし。

楽曲リスト:

## 1. プロンプト構造パターン

### 1-1. 極端な「簡素プロンプト」型
TenJR/CK/音色とは**正反対**。プロンプトは数個のジャンルタグのみで、Sunoの自動解釈に大きく委ねる。

| 曲 | プロンプト全文 |
|---|---|
| 間 | (空) |
| ランジェリーさま | `drum and bass` |
| 国語辞典に載らない国 | `slower tempo, acoustic focused, minimal arrangement` |
| そばいてよ | `indie, spoken word, drum and bass` |
| 夜の底から | `anthemic, rock, pop rock` |
| 世話人さん | `pop rock, punk, indie` |
| こころの声 | `punk, electric guitar, acoustic, electric, female vocals` |
| 器 | `upbeat melodic hardcore, singalong, positive` |

→ 12曲中11曲が**1〜10タグ程度**のミニマル指示。これが最大のシグネチャ。

### 1-2. ジャンルタグの「重ね合わせ」型
複数ジャンルをカンマで並べるが、説明や形容詞は付けない。

- `acoustic guitar, acoustic, electric guitar, indie pop, pop, japanese, rap, indie, electric` (最後の恋)
- `danceable, playful, acoustic, rap, pop, rock, pop rock, indie pop` (Ladybug)

→ 同一カテゴリの近接タグを冗長に重ねる (acoustic, electric guitarとelectricを別途記述)。Sunoの解釈精度を上げる狙いか、あるいは"ジャンル横断"を演出する装置。

### 1-3. BPM指定はほぼ無し
12曲中、BPM明示は **1曲のみ** (もしも君が泣く前に〜: BPM 175-185)。残り11曲は未指定。

→ テンポをAIに委ねる設計。

### 1-4. キー指定は完全に無し
12曲全てキー指定なし。

### 1-5. ボーカル指示も最小限
明示があるのは2曲のみ:
- `female vocals` (こころの声)
- `shouty male vocals` (正しいと楽しいどっちが良いですか)

→ 性別とテンションを示すのみで、トーンや質感の細分化は無い。

### 1-6. Persona/外部参照型
`間.txt` のNOTESに `Persona レフティートニーを使用` と記載。Sunoのカスタム声・Personaを直接呼び出す方式。

→ プロンプトを書かない代わりに、保存済みPersonaを参照する**省力化テクニック**。

### 1-7. ネガティブプロンプトは皆無
12曲全てでネガティブ指示なし。

→ TenJR (`No studio polish`)、CK (`reduced to minimal`)、音色 (`Not Sexy, Not Deep`) と比較して、ランジェリーは**完全に肯定指示のみ**。

## 2. 歌詞構造パターン

### 2-1. 「自由律詩」型 + 全角【】マーカー
構成マーカーは `【Aメロ】【Bメロ】【サビ】【Cメロ】【ラスサビ】【アウトロ】` を採用。CKの英語シネマ台本型とは異なる、J-Pop伝統に沿った日本語マーカー。

→ 12曲中、約半数で全角【】を使用。残りは無マーカー or [Verse]英語マーカー。

### 2-2. 詩的・自由律・比喩多重層の長文歌詞
ほとんどの曲が **30行以上**、行ごとに比喩や意味のレイヤーを重ねる。

例 (国語辞典に載らない国):
```
炎上は無料　共感は有料
優しさはサブスク制
本音は規約違反で
建前だけが上場中
```

→ 4行で4つの**現代社会への風刺メタファー**が連射される密度。

### 2-3. 「ランジェリー」モチーフの重層使用
全12曲中、5曲以上で「ランジェリー」が登場。意味が**曲ごとに変わる**。

| 曲 | ランジェリーの意味 |
|---|---|
| ランジェリーさま | 性的欲望/神秘/真理の象徴 |
| 夜の底から | アーティスト名のサインオフ |
| Ladybug | 自由/秘密/夢の象徴 |
| こころの声 | シュール小道具 (惑星間ワープの切符) |
| 最後の恋 | (タイトルだけで本文には登場せず別キーワード「こころのこり」) |

→ 同一モチーフを**メタファーの層を変えて使い回す**手法。アーティスト固有の象徴体系。

### 2-4. シュール日常スケッチ型
意味の通る現実描写と、超現実的なナンセンスを並列配置。

例 (こころの声):
```
ランジェリー干した焼きそばパン　午前5時の君主論
「今日は右と左、どっちを外す？」
心のドローンが北極を周回中
ゾウの滑り台にフリルのダイブ
```

→ 通常の意味解釈を破壊する**ダダ的言語遊び**。

### 2-5. 社会批評型
日本社会への直接批評。「正しさ」「平均」「同調圧力」への懐疑が中心テーマ。

代表例:
- 国語辞典に載らない国: `平均値が神様で / はみ出た君は不正解`
- 正しいと楽しいどっちが良いですか: `正しい事と楽しい事 / どっちが良いですか？`
- そばいてよ: `「普通」って何？と　窓に問いかけた`
- 器: `争いの種は　いつでも些細なすれ違い`

→ 設問形式 (「どっちが良いですか？」) と命題形式 (「〜は〜だ」) を併用。

### 2-6. 「対句構造」の多用
2行 or 4行の並列構造で対比を示す。

- `炎上は無料　共感は有料 / 優しさはサブスク制 / 本音は規約違反で / 建前だけが上場中` (国語辞典)
- `間違えない人生と / 考えない人生 / どっちが怖いか` (国語辞典)
- `誰と誰の間で世界はつくられるの / 先生と生徒の間　友達と嘘の間 / 君と僕の間には　見えない壁と橋がある` (間)

### 2-7. 韻は緩やか — 意味とリズム優先
音色やTenJRのような語末モーラ完全一致は少ない。代わりに**意味の対応**と**音節リズム**で詩的快感を作る。

例 (もしも君が泣く前に〜): 比較的密度の高い韻
```
正論安全圏平均点
全部並べて非常識宣言
君の沈黙一秒延長
それだけで僕は全速前進
```
→ 「点／宣言／延長／前進」、「-en」「-ou」音の連射。これはランジェリー作品の中では**例外的に韻密度が高い**。

### 2-8. 英語フレーズの混入は控えめ
TenJR/音色のような積極的コードスイッチは少ない。英語が多いのは2曲 (器、そばいてよ)。

- 器: `HAND IN HAND! / SIDE BY SIDE! / This is our anthem` (アンセム強調用途)
- そばいてよ: `Stand by me` (引用)

### 2-9. 自分・他者の二人称多用 (君/僕)
「君」「僕」が中核。**閉じた二人世界**の歌詞が多い。

- 「君と僕」「君のそばに」「君が泣くなら」「君のランジェリー」

→ 一人称的内省+対話的呼びかけが基調。
