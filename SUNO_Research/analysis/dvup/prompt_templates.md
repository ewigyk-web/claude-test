## 重要: dvupを再現するチェックリスト

- [ ] 参照アーティスト (X JAPAN / BABY METAL / 黒夢 / V系金字塔等) を1人決めたか
- [ ] NOTES欄に「○○風味」と記載したか
- [ ] プロンプトを「ライブ感タグ + ジャンル本体」の2段構造にしたか
- [ ] BPMを範囲指定で書いたか (単一値より柔軟)
- [ ] メタルサブジャンルを5-10個列挙したか
- [ ] ボーカルを4タイプから1つ選んだか
- [ ] 楽器をプレイヤー視点で書いたか (twin lead / palm-muted / down-tuned等)
- [ ] オーケストラ要素 (strings/choir) を入れたか (シンフォニック系の場合)
- [ ] 歌詞テーマを「闇→再生」のメタル王道アークに沿わせたか
- [ ] メタル王道英語語彙 (Fire/Flame/Wings/Eternity等) を使ったか (英語曲)
- [ ] V系日本語の耽美漢字を使ったか (V系曲)
- [ ] サビでタイトル/キーフレーズを4-6回反復したか

## 流用Tips

- **参照アーティストの選び方**: 自分の好きな/詳しいメタルアーティストを5人ほどリスト化し、曲ごとに1人選ぶ。
- **2段プロンプトの効果**: ライブ感タグを最初に置くと、Sunoがステージング/空間感を意識する。
- **ジャンル列挙の上限**: 10タグ程度が上限。それ以上は解釈ブレが大きくなる。
- **ネガプロほぼ無し戦略**: dvupはネガを書かない。代わりに**ジャンルタグの精度**で方向を絞る。
- **異色実験**: メタル枠内なら何でも試せる。Loli Pop / Middle Eastern / Black Comedy など、他ジャンル要素を混ぜる勇気を持つ。
