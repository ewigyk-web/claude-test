# アーティスト比較 — CK

## CK — 全体像

| 項目 | 傾向 |
|---|---|
| 主軸ジャンル | Djent / Progressive Metal / Industrial |
| 副ジャンル | Cyberpunk Synth, Dark Pop, Idol Pop (kawaii), City Pop |
| BPM帯 | 140-155 (狭い帯域に集中) |
| Key | ほぼ全曲 Minor (B minor / C# minor 等) |
| ボーカル | 男女両方使用、セクション別に "whisper ↔ belt/scream" の対比 |
| 歌詞言語 | ほぼ英語のみ (一部、Almost Connectedはテーマのみ日本語) |
| 構造指示 | [Section名 - 演出ディレクション] のシネマ台本形式 |
| ネガプロ | 「No XXX」より「reduced to / minimal / sparse」の減算表現 |
| 高品質マーカー | NOTES欄に短い感想 (例: 「名曲」「良い曲」「Remix済」) |

## CKのクセ・強み

強み:
- **ジャンル合成の精度**: 必ず2-3ジャンルを × でクロスし、独自音像を作る。
- **シネマ台本型のプロンプト**: Intro/Verse/Chorus/Bridge/Outro 各々に演出を書く。これがCKの最大の武器。
- **囁き ↔ 爆発のダイナミクス対比**を全曲で徹底。
- **沈黙を構造に組み込む**: `(silence)` `(no reply)` を歌詞に明文化する独自手法。
- **デジタル語彙の世界観統一**: signal/data/glitch/code… で曲を貫通させる。

クセ:
- BPM/Keyが狭く均質化 (140-155 / Minor)。アップテンポやMajor曲が無い。
- ほぼ英語歌詞のみ。日本語表現は希薄。
- ネガティブプロンプトが弱い → AIの暴走を抑える保険が少ない。
- セクション指示が複雑なため、Sunoが解釈を誤る可能性がある（要検証）。

## TenJRとの対照 (参考)

| 観点 | TenJR | CK |
|---|---|---|
| プロンプト書式 | 長文記述型 | 短タグ列挙 + シネマ台本 |
| ジャンル | デジタルハードコア中心 | Djent × Cyberpunk × Industrial |
| BPM | 200+ / 90台 の二極化 | 140-155 に集中 |
| 歌詞言語 | 日英コードスイッチ | 英語中心 |
| ダイナミクス | 終始ハイテンション | 囁き ↔ 爆発の対比 |
| ネガプロ | 強い (No XXX 連発) | 弱い (減算表現) |
| 世界観 | 物理的轟音・ライブ感 | デジタル疎外・孤独 |

→ **TenJR=肉体派、CK=精神派**という対比軸で捉えられる。
