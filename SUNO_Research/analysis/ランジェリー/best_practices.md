# ベストプラクティス — ランジェリー

## 1. プロンプトは「最小限のタグ列挙」で済ませる

| こう書くと | こうなる |
|---|---|
| `drum and bass` | DNBの標準的アレンジが立ち上がる |
| `upbeat melodic hardcore, singalong, positive` | アンセム系メロディックHC |
| `acoustic guitar, indie pop, japanese, rap` | 弾き語り風J-Indie + 軽いラップ要素 |

→ 細かく書かないことで**歌詞の自由度**が上がり、AIが歌詞のリズム/感情に合わせやすくなる。

## 2. PROMPT空 + Persona利用で完全省力化

```
PROMPT: (空)
NOTES: Persona [自前のPersona名] を使用
```

→ 自分の声/スタイルを保存済みPersonaで呼ぶ方式。プロンプト記述を完全に省略可能。

## 3. ジャンルタグを「冗長に」重ねる

```
acoustic guitar, acoustic, electric guitar, indie pop, pop, japanese, rap, indie, electric
```

→ 同じカテゴリのタグを敢えて重複させる (`acoustic` と `acoustic guitar`)。Suno側で多面的に解釈される。

## 4. 構成マーカーは全角【】で日本語表記

```
【Aメロ】
【Bメロ】
【サビ】
【2番Aメロ】
【Cメロ】
【ラスサビ】
【アウトロ】
```

→ J-Pop伝統的な構造マーカー。日本語歌詞には半角[Verse]より馴染む。

## 5. アーティスト固有モチーフを「多義シンボル」として使い回す

ランジェリーは「ランジェリー」を以下の意味で使い分ける:
- 性的欲望/真理 (ランジェリーさま)
- 自由/秘密 (Ladybug)
- ファンタジー象徴 (こころの声)
- アーティスト署名 (夜の底から)

→ 1つのキーワードを複数曲で**意味を変えて使う**ことで、世界観の連続性が生まれる。

## 6. 「対句連打」で社会批評の密度を上げる

```
炎上は無料　共感は有料
優しさはサブスク制
本音は規約違反で
建前だけが上場中
```

→ 4行並列で4つの風刺を連射。**現代用語 (サブスク/規約違反/上場) を使うと刺さる**。

## 7. 設問形式のフックで「巻き込む」

```
正しい事と楽しい事
どっちが良いですか？
```
```
ここが日本の未来なら
僕はいぶつでいたい
```

→ 命題を断定するのではなく、**読者に問いを投げる**。タイトルそのものが疑問文の例。

## 8. シュール日常スケッチの作り方

```
ランジェリー干した焼きそばパン　午前5時の君主論
心のドローンが北極を周回中
ゾウの滑り台にフリルのダイブ
```

公式: `日常物 + 異種混合 + 突飛な動詞`
- 「焼きそばパン」(日常) + 「君主論」(異種) を「干した」(突飛) で接続
- 意味よりも**音と画像のインパクト**を優先

## 9. 「君と僕」の閉じた二人世界

```
君と僕とでつなぐ物語
そのすき間で息してるいのち
```

→ 大きな主語 (社会/世界) を語った後、必ず「君と僕」に降ろして締める構造。

## 10. 高速ラップが必要な時だけ詳細指定

通常の楽曲は簡素プロンプトで十分だが、**ラップが本格的に必要な曲のみ**詳細を書く:

```
rap, Japanese alternative rock rap, modern J-rock, fast tempo, BPM 175–185,
emotional, intense, minimal, emotional, strong, rapid, rhythmic,
alternative rock, j-rock, rock
```

→ BPM明示 + 速度形容詞 (fast/rapid/rhythmic) を重ねる。これが12曲中唯一のBPM指定例。

## ランジェリー流の哲学

> **「歌詞を書け。プロンプトは添えるだけでいい。」**

TenJR/CK/音色がプロンプトで音像を作り込むのに対し、ランジェリーは**歌詞そのものが楽曲の骨格**。AIに音響面を委ね、自分は詩人としての仕事に集中するスタイル。

このアプローチが向く人:
- 詩・歌詞を書くのが得意
- 細かい音色指定より自由度を重視
- ジャンルを跨いで色んな楽曲を試したい
- プロンプトに時間をかけたくない
