| 項目 | 傾向 |
|---|---|
| 主軸ジャンル | J-Rock / Hard Rock / Metal |
| 副ジャンル | 和楽器系、Britpop、Punk、Krautrock、DnB、Industrial Gothic |
| BPM帯 | 部分的指定 (140 / 155)、その他は未指定 |
| Key | 全曲未指定 |
| 楽器シグネチャ | **歪みギター + 和楽器 (shamisen/taiko)** の融合 |
| ボーカル | 男女 / Twin / 英日 / Denpa まで多様 |
| 歌詞言語 | 日本語/英語/混合の3パターン使い分け |

| プロンプト密度 | **過剰**。1曲5-7ジャンル並列、シネマ的進行記述 |

## Meikai Soulのクセ・強み

強み:
- **過剰ジャンルクロスの実装力**: 1曲に5-7ジャンルを並列指定し、しかも破綻させない構成感覚。
- **和楽器ロックの探求**: shamisen/taiko/shakuhachi/koto を主役級にロックと融合。「No guitars, no synths」(糸の影)で和楽器のみ完結も可能。
- **シネマ的アレンジ進行**: Verse→Pre→Chorus→Bridge→Final の各段階で音響変化を文章で詳細指定。
- **ボーカル多様性**: 男性 grit / 女性 ferocious / twin / Denpa / 英語 / 日本語 を曲ごとに使い分ける守備範囲。
- **王道直球の歌詞**: 難解さを排し、「夜→光」「諦めない」の伝統的ロック語彙を直接ぶつける。
- **シリアス↔コメディの振れ幅**: 「夜を越えた光」の重厚と「No Coffee」のドタバタが同居。
- **NOTES活用**: 自作評価を1行記録するアーティスト独自の運用。

クセ:
- プロンプトの専門度が高すぎ → 一部タグ (post-rock textures, denpa-style 等) はSuno解釈が安定しない可能性。
- BPM/Key未指定が大半 → 同曲再生成時の揺らぎ大。
- 歌詞テーマが王道すぎ → kedaruge/ランジェリーのような独自性は薄い。
- 11曲中6曲が「夜・光・闇」モチーフ → 世界観としての変奏は要工夫。
- 過剰ジャンルクロスは**Sunoが処理しきれない場合がある**。

## Meikai Soulを位置付ける一行

> **「ロック総合商社。あらゆる派閥を1人で回し、和洋を貪欲に融合させる。」**

他5者が特定の哲学/美学に絞るのに対し、Meikai Soulは**全方位ロック**。J-Rock/Metal/Britpop/Punk/和楽器/Krautrock/Industrial を一人で網羅する。武器は「過剰なジャンル合成力」と「歌詞の王道直球」。
