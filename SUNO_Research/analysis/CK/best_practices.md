# ベストプラクティス — CK

## 1. ジャンルは「× で2-3個クロス」する

| こう書くと | こうなる |
|---|---|
| `Djent × Celtic Folk × Classical Piano` | 重い刻みに弦と鍵盤が乗る独自音像 |
| `Industrial Metal × Dark Orchestral × Experimental J-Rock` | 機械的+管弦楽+J-Rock感の三層 |
| `City Pop × Cyberpunk Synth` | 夜の都市感 + 電子的な冷たさ |

→ 単一ジャンル指定より独自性が出る。3個までが上限。

## 2. セクション別に「演出ディレクション」を書く

```
Intro: fragile whisper + reversed piano, almost empty space
Verse: jazzy dark piano with loose groove, calm but eerie
Pre-Chorus: tension rises → abrupt cut (near silence)
Chorus: violent drop — djent guitars, blast beats, deep sub
Breakdown: mechanical hits, distorted bass pulse → explosive scream
```

→ 全体音像をまとめて書くより、セクション別に書くほうがダイナミクスが立つ。映画の脚本のように書く。

## 3. ボーカルはセクション別に対比させる

| Section | 指示例 |
|---|---|
| Verse | `breathy sultry low alto, soft airy whisper tone, intimate and controlled` |
| Chorus | `explosive powerful belting, edgy emotional release, strong melodic hook` |
| Bridge | `whisper-dark close vocal, tense minimal delivery` |
| Outro | `fading whisper` |

→ "囁き → ベルト → 囁き"の対比を全曲で設計する。

## 4. 歌詞に「沈黙」を書く

```
[Twist (Silence)]
(…silence)

if I stop first
…would you notice?
```

→ `(silence)` `(no reply)` を歌詞段階で明文化すると、AIが本当に間を作る。

## 5. グリッチ表現は「単語を分断 + 反復」で作る

```
don't re—reset me
don't re—reset me
I repeat you
I repeat you
```

→ ハイフンや「…」で単語を分断し、同フレーズを2-4回反復する。サビフックがエフェクトと一体化する。

## 6. ネガティブは「禁止」より「最小化」で書く

| 悪い書き方 | CK風の書き方 |
|---|---|
| `No EDM, no flashy synth` | `EDM elements reduced to subtle atmospheric layer` |
| `No bright synth lead` | `no flashy synth leads, minimal "sparkle"` |
| `No busy arrangement` | `Sparse Arrangement` |

→ 「あるけど抑えめ」が出る。完全排除より自然な仕上がり。

## 7. メタヘッダー型プロンプトでテーマを伝える

```
TITLE : Glass City
THEME : 夜の都市・孤独・切なさ／癖ある女
VOCAL : Half-Sung Half-Spoken, Smooth Emotional, Mellow
BPM   : 150 / KEY : Minor
```

→ ラベル形式でAIに"設定資料"を読ませる。物語性のある曲で有効。

## 8. デジタル語彙バンクで世界観を統一

頻出語: `signal, frequency, static, glitch, code, data, screen, wire, pixel, neon, system, reset, refresh, feed, ghost, machine, layer, frame, sync, capture, fracture`

→ 1曲に5-7語を散りばめると、楽曲群がアルバムとして繋がって聴こえる。

## 9. アウトロは「囁き or スポークン」で締める

```
[Outro] (fading whisper)
I always liked you...
always...
```

```
[Outro — Spoken, Glitch Fade]
I could've gone home earlier.
I didn't want to.
Don't read into it.
```

→ サビをそのまま繰り返さず、独白に降ろして余韻を作る。

## 10. BPMはMinor Keyの140-155にハマる

CK全曲の集中帯。Cyberpunk/Djent系を作るなら、まずこのレンジを試す。
- Slow & Heavy寄り → 140
- Mid Aggressive → 150
- Tense & Dark → 155
