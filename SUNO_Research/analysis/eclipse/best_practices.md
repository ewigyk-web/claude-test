# ベストプラクティス — eclipse

## 1. プロンプトを書かない勇気を持つ

eclipseの8曲中5曲がプロンプト空 or 1-3単語のみ。

→ 自分の歌詞とSunoの自動解釈を信頼する。**「迷ったら空のまま投げる」**勇気が一つの戦略。

## 2. ネガティブのみで方向指定

```
‑Growl, ‑Metal, ‑Lo-fi, ‑Distorted, ‑Male Vocals, ‑Dark, ‑Aggressive
```

→ ポジティブ指示を一切書かず、**ネガティブだけで「優しい/明るい/女性Vo/クリーン」**を間接指定する逆説手法。

「明るい曲が欲しい」=「暗くないものが欲しい」=「-Dark を書く」。

## 3. キーワード単独プロンプト

```
electric sitar
Trill/Rolled R
```

→ **特殊な楽器/技法を1つだけ強調**し、残りはAIに委ねる。曲の個性を1要素で出す。

## 4. 信頼できる共通ネガリストを流用

HETEROCHROMIAから流用したリスト:
```
‑300Hz-500Hz mud, ‑boxiness, ‑phase cancellation, ‑monophonic sub,
‑frequency masking, ‑vocal gasps, ‑humming, ‑artifacts, ‑hiss, ‑noise
```

→ 自前で書かなくても、**他者のテンプレを借りて品質保証**する。

## 5. 詳細プロンプトは「特殊技法」がある時だけ書く

詳細を書く2例外曲はどちらも特殊技法が中核:
- Whisper Me Closer → `slap guitar` `Steel-string Acoustic` `Melodic double bass`
- Ephemeral Bloom → `Hiccup Vocals` `Whispered Vocals` `Falsetto` `Celtic music`

→ 通常曲はプロンプト極簡、**特殊技法を使う曲だけ詳細化**するメリハリ。

## 6. 文学作品オマージュを活用

- アルジャーノンに花束を → ダニエル・キイス小説
- Ephemeral Bloom → 環境終末ファンタジー

→ 既存の物語/文学から発想を借りる。**ゼロから世界観を作るより効率的**で、深みも出る。

## 7. 「劣等感→自己肯定」のアーク

eclipseの王道歌詞構造:
```
Verse: 比較社会で傷つく自我
   ↓
Pre-Chorus: 違和感の蓄積
   ↓
Chorus: 「それでも僕は」の宣言
   ↓
Outro: 静かな自己肯定の余韻
```

例 (朝は来る):
- 起: `周りと比べて息を潜めて`
- 結: `いつか羽ばたく / 僕の空へと向かうため`

→ 現代SNS時代の**比較疲れ**を起点に、自己肯定で締める。

## 8. 日英対訳併記

3曲で実装。形式:

```
{英語フル歌詞}

🐧Japanese lyric
{日本語フル歌詞}

⭐日本版です～
{再度の日本語歌詞}
```

→ Sunoの再生成時に英/日どちらでも歌わせられる。音色と同じ手法。

## 9. 「畳みかけ4連反復」で意識崩壊を表現

忘却のロンドの特徴的手法:
```
回る 回る 回る 回る 記憶
消える 消える 消える 消える 世界
```

→ **同語4連反復**で催眠的/崩壊的な印象を作る。歌詞の意味より音響リズムを優先。

## 10. 擬音アウトロ

Whisper Me Closer:
```
tunk… hush… mmh… stay… ahhh…
tunk… hush… mmh… stay… mmh…
```

→ 親密な接触音を**擬音化**して終わらせる。言葉より体感の余韻。

## 11. 日英コードスイッチはピンポイント

Daraと同じく、英語は**1〜2行の決定的命題**として刺す:

```
WHY AM I SO WEAK?! ひび割れた鏡
The gap between us, it's tearing me inside
Still I breathe… I'm here
I'm not lost, I'm not broken
```

→ 英語は装飾でなく**刃のように**刺さる位置に配置。

## 12. アウトロは「囁きで消える」

頻出パターン:
```
アルジャーノン、おやすみ
Sayonara... my dear spring...
still… still… I remain
tunk… hush… mmh… stay…
```

→ 大声で終わらず、**消える/呟く**形でフェード。HETEROCHROMIAと類似。

## 13. 比較社会のテーマで等身大を書く

頻出語彙:
- 比べる / 周りと / 違う / 弱い
- 影 / 光 / 空 / 鳥 (羽ばたく対象)
- 鎖 / 縛り / ラベル

→ SNS時代の**比較疲れ**を直接的に書くことで、現代人に響く歌詞になる。

## 14. ギタリストとしての自己認識を持つ

NOTESで自作評価:
- `流石ギタリストだけあってギターが上手い`
- `ギターの音質が見事！`
- `巻舌レイヤードヴォーカル スラップギター`

→ 「自分はギタリストだ」という自己認識が、**プロンプト省略の自信**に繋がる。

## 15. プロンプトを書くスタイルを楽曲ごとに変える

eclipse は1人で5パターンのプロンプトスタイルを持つ:
1. 完全に空
2. 1単語のみ
3. ネガティブのみ
4. 共通ネガリスト流用
5. 詳細記述

→ **曲の性質に応じてスタイルを切り替える**。テンプレを統一しないのも一つの戦略。

## eclipse流の哲学

> **「ギターが弾ければプロンプトはいらない。歌詞で自分を曝け出せ。」**

プロンプトを書き込む他のアーティストと違い、eclipseは**自分のスキル**を信頼してプロンプトを最小化。代わりに歌詞には**等身大の自分**を曝け出し、文学オマージュを通して深みを出す。

このアプローチが向く人:
- 自分の楽器スキル (特にギター) に自信がある
- プロンプト記述に時間をかけたくない
- 比較社会で傷ついた経験がある
- 文学/物語が好きで、それを歌詞にしたい
- Sunoの自動解釈を信頼できる

**注意:**
- BPM/Key未指定が多いので、再生成時の揺らぎは諦める
- 楽器スキルがない場合、この手法は危険 (Sunoが標準解釈に流れて凡庸になる)
- ネガティブのみ戦略は「明確に避けたいもの」がある時だけ有効
